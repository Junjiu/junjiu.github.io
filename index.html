ethical
2018-11-27
Uncategorized

When I was taking SE319, there was a project, which doesn’t have
some specific requirement, only requiring use JavaScript. I had
done some javaScript project before, but if I submit them, it may
cause ethical dilemmas. So I asked instructor whether I can use
the original code, which meet all the requirement of this project,
he said no. So I made a totally new project, and learn a lot.
I think when facing ethical dilemmas, the best solution is asking
some have experience, or just ask the one who will judge it later.
I think in our daily working, there will be many chance to reuse our
code. I think there are two point that we should take care. First,
we should know who is the code’s owner. Do we have the
ownership of the code, if so, we should be able to use it. Second,
the purpose of using the code is also very important. If it is for
academic reason, we must consider the academic dishonesty.
Anyway, adding a citation will always be the right thing to do.
